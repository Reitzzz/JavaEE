# 项目报告：智能图书借阅管理系统

## 1. 项目简介

本项目基于 Spring Boot 3 开发，面向图书馆或班级资料室的图书借阅场景，实现用户登录、角色权限、图书分类、图书维护、借阅归还和 AI 图书推荐功能。

## 2. 核心实体与数据表

核心实体包括用户、角色、图书、分类、借阅记录。数据库共 6 张表：

- `users`：系统用户
- `roles`：角色
- `user_roles`：用户角色关联
- `categories`：图书分类
- `books`：图书
- `borrow_records`：借阅记录

## 3. 业务流程

读者登录后可以查看图书、搜索图书、借阅图书、归还自己的借阅记录。管理员登录后可以维护分类和图书，并查看全部借阅记录。

## 4. 创新功能

系统提供 AI 图书推荐助手，后端通过 OpenAI 兼容的大模型接口提交当前馆藏数据和用户问题，让大模型根据真实馆藏生成推荐结果。未配置 API Key 时，系统会自动切换到本地规则推荐，保证演示流程可用。

## 5. 部署说明

先创建 MySQL 数据库 `smart_library`，再修改 `application.yml` 或设置环境变量 `DB_URL`、`DB_USERNAME`、`DB_PASSWORD`。如需启用大模型功能，设置 `LLM_BASE_URL`、`LLM_API_KEY`、`LLM_MODEL`。最后在项目目录执行 `mvn spring-boot:run`，访问 `http://localhost:8080`。

## 6. 难点与解决方案

- 权限控制：使用 Spring Security，将管理员和读者权限分离。
- 借阅一致性：借阅和归还使用事务，避免库存数量和借阅记录不一致。
- 大模型接入：使用可配置的 OpenAI 兼容接口，避免密钥写死在代码中。
